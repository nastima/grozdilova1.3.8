# description

```
В этом задании нужно реализовать небольшую библиотеку для управления коллекцией книг. Тебе необходимо создать класс LibraryCollection с методами для добавления книг, их удаления, получения информации о книгах и количестве книг в коллекции.
Затем напишите тесты, используя Vitest, чтобы проверить корректность работы реализованных методов.

addBook(title: string, author: string): string | Error - Добавляет книгу в коллекцию, возвращает id книги. Если книга с таким названием уже существует, возвращает ошибку.
removeBook(id: string): void - Удаляет книгу по id книги.
getBookInfo(id: string): { title: string, author: string } | null - Возвращает информацию о книге по id.
getAllBooks(): Array<{ id: string, title: string, author: string }> - Возвращает список всех книг.
getBooksCount(): number - Возвращает количество книг в коллекции.

```

### запуск проекта

`npm i`
`npm run test`
